History

Javascript was originally called LiveScript when it was first released. It was developed by Netscape Communications. Netscape Communications had a the goal of integrating java into their browser to make it more dynamic. Instead they decided to create a scripting language that was similar to java. And three months after the release 0f LiveScript, in December 1995, Javascript was officially released.
